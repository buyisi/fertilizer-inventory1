# 化肥农药库存管理系统

## 项目概述
本项目是一个简单的化肥农药库存管理系统，用于管理化肥和农药的入库、出库、库存跟踪和过期提醒。系统支持条码扫描功能，可以使用手机或电脑扫描器进行操作。

## 功能需求

### 1. 入库管理
- 条码扫描入库
- 手动输入产品信息
- 记录生产日期和过期时间
- 自动更新库存数量

### 2. 出库管理
- 条码扫描出库
- 手动输入出库数量
- 自动更新库存数量

### 3. 库存管理
- 查看当前库存水平
- 按产品类型筛选
- 查看产品详细信息

### 4. 过期提醒
- 显示即将过期的产品（30天内）
- 按过期日期排序

### 5. 条码扫描
- 支持手机摄像头扫描（需要HTTPS）
- 支持电脑外接扫码器

## 技术要求

### 前端
- HTML5 + CSS3
- EJS模板引擎
- ZXing-js库（条码扫描）
- 响应式设计（支持移动端和桌面端）

### 后端
- Node.js
- Express框架
- SQLite数据库

### 部署
- 本地部署
- Vercel部署（用于HTTPS支持）

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动服务器
```bash
npm start
```

### 3. 访问系统
- 本地访问：http://localhost:3000
- 手机访问：使用同一网络下的电脑IP地址，例如 http://192.168.1.100:3000

## 部署到Vercel
1. 将代码推送到GitHub仓库
2. 在Vercel上导入仓库
3. 完成部署后，使用Vercel提供的HTTPS域名访问系统

## 注意事项
- 手机摄像头扫描需要HTTPS环境，本地部署可能无法使用此功能
- 系统使用SQLite数据库，数据存储在本地文件中
- 为了获得最佳体验，建议部署到Vercel获取HTTPS支持
